# Data Directory

Данная директория содержит все данные проекта.

## Структура

- `raw/` - Исходные необработанные данные (законодательные тексты, документы)
- `processed/` - Обработанные и структурированные данные (блоки, графы)
- `samples/` - Примеры данных для тестирования и демонстрации
- `models/` - Обученные ML модели и эмбеддинги

## Примечания

- Файлы в `raw/` и `processed/` не коммитятся в git (см. .gitignore)
- Примеры в `samples/` коммитятся для целей тестирования
- Модели могут быть большими, загружаются отдельно
